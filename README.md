# EXIF情報表示ツール

画像ファイルからEXIF情報を抽出し、網羅的に表示するWebアプリケーションです。

## 機能

- **対応ファイル形式**: JPEG、PNG、TIFF
- **ドラッグ&ドロップ対応**: 簡単に画像をアップロード
- **カテゴリ別表示**: 基本情報、カメラ情報、撮影設定、GPS情報、その他
- **検索機能**: EXIF情報を検索・フィルタリング
- **生データ表示**: JSON形式で全データを確認
- **GPS地図リンク**: GPS情報がある場合、Google Mapsで位置を表示
- **プライバシー配慮**: すべての処理はクライアントサイドで実行（サーバーへのアップロード不要）

## 使い方

1. `index.html` をWebブラウザで開く
2. 画像をドラッグ&ドロップ、またはクリックしてファイルを選択
3. 自動的にEXIF情報が表示されます
4. タブを切り替えて各カテゴリの情報を確認
5. 検索ボックスで特定の情報を検索可能
6. 「生データ（JSON）表示」をチェックすると、JSON形式で全データを確認できます

## 技術仕様

- HTML5
- CSS3
- Vanilla JavaScript
- EXIF.js ライブラリ（CDN経由）

## 注意事項

- PNG形式は通常EXIF情報を保存しないため、多くの場合「EXIF情報なし」と表示されます
- ファイルサイズの上限は10MBです
- すべての処理はブラウザ内で完結し、サーバーにデータは送信されません

## 対応ブラウザ

モダンブラウザ（Chrome、Firefox、Safari、Edge）の最新バージョンで動作します。
